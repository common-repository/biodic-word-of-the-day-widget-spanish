== Description ==

BioScripts - Centro de Investigación y Desarrollo de Recursos Científicos ha desarrollado este plugin para la difusión del proyecto BioDic que te permite tener en tu sidebar la palabra del d&iacute;a que se muestra en su web: https://www.biodic.net y as&iacute; aprender cada d&iacute;a una palabra nueva. Tan s&oacute;lo tienes que arrastrar el widget a tu sidebar y mostrar&aacute; la definci&oacute;n completa de la palabra del d&iacute;a de BioDic.


== Installation ==
1. Sube el archivo a wp-content/plugin
2. Activar plugin
3. Ir a Apariencia > widgets
4. Arrastra el widget al sidebar que quieras.
5. &iexcl;Comprueba que se ve correctamente!
6. &iexcl;Dale un poco de diseño!
